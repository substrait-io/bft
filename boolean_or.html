Or

Defined in functions_boolean.yaml

The boolean `or` using Kleene logic. This function behaves as follows with nulls: true or null = true null or true = true false or null = null null or false = null null or null = null In other words, in this context a null value really means "unknown", and an unknown value `or` true is always true. Behavior for 0 or 1 inputs is as follows: or() -> false or(x) -> x
